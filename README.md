### Dependencies
Supported OS: Windows 7+

Libraries:
* [GLFW 3.3.2](https://www.glfw.org/)
* [GLM 0.9.9.8](https://github.com/g-truc/glm)
* [FreeType 2.12](https://freetype.org/)
* [FFTW 3.3.10](https://www.fftw.org/)
* Winmm and Windows.h (for microphone input)
